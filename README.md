# 📊 Project Overview

A comprehensive **Exploratory Data Analysis (EDA)** of Amazon Prime Video's streaming content library, analyzing over **9,000+ titles** and **124,000+ cast and crew records** to uncover strategic insights about content distribution, quality metrics, regional patterns, viewer engagement, genre preferences, and platform growth opportunities.

This project aims to transform raw entertainment data into meaningful business insights that can support decision-making in content acquisition, audience engagement, regional expansion, and production strategy.

The analysis explores:

✨ Content Distribution Patterns

✨ Genre Popularity Trends

✨ IMDb & TMDB Performance Metrics

✨ Production Country Analysis

✨ Viewer Engagement Indicators

✨ Movie vs Show Comparison

✨ Content Growth Trends

✨ Cast & Crew Contributions

✨ Popularity and Rating Correlations

✨ Strategic Recommendations for Platform Growth

# 🎯 Business Objectives

## 🎬 Content Strategy

Identify dominant genres and content gaps for acquisition decisions and future content investments.

### Key Questions:

* Which genres dominate Amazon Prime Video?
* What content categories are underrepresented?
* Where should future investments be focused?

---

## ⭐ Quality Analysis

Understand content performance through IMDb and TMDB ratings.

### Key Questions:

* What is the average quality of content on the platform?
* Which genres receive the highest ratings?
* How do ratings vary across content types?

---

## 🌍 Regional Expansion

Analyze production country distribution for market growth opportunities.

### Key Questions:

* Which countries contribute the most content?
* Are there opportunities in emerging entertainment markets?
* How diversified is the content library geographically?

---

## 📺 Platform Optimization

Compare movie versus show performance to understand audience preferences.

### Key Questions:

* Do viewers engage more with movies or shows?
* Which content type receives better ratings?
* What content mix should be prioritized?

---

## 📈 Trend Identification

Track content additions and popularity patterns over time.

### Key Questions:

* How has the content library evolved?
* What trends are emerging in content production?
* Which periods experienced significant growth?

# 📁 Dataset Description

## 🎥 Titles Dataset (9,000+ Records)

**File:** `titles.csv`

The titles dataset contains detailed information about movies and television shows available on Amazon Prime Video.

### Included Features:

✔ Title Information

✔ Content Type Classification

✔ Release Year

✔ Genre Categories

✔ Production Countries

✔ IMDb Ratings

✔ IMDb Vote Counts

✔ TMDB Ratings

✔ TMDB Popularity Scores

✔ Runtime Information

✔ Age Certifications

---

## 👥 Credits Dataset (124,000+ Records)

**File:** `credits.csv`

The credits dataset contains cast and crew information associated with Amazon Prime Video content.

### Included Features:

✔ Actor Information

✔ Director Information

✔ Character Names

✔ Role Classification

✔ Title Associations

✔ Cast Participation Analysis

✔ Director Contribution Analysis

✔ Talent Distribution Insights
